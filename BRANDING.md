# Chronix Brand Usage Policy

## Overview

"Chronix" is the brand name for this software project. While the source code is licensed under the GNU Affero General Public License v3.0 (AGPL-3.0-only), the Chronix name and logo are brand identifiers that require separate consideration for use.

## Permitted Uses

You **may**:

- Use "Chronix" to accurately describe or reference this software (nominative use)
- State that your product or service "works with Chronix" or "is compatible with Chronix"
- Describe your project as a "fork of Chronix" or "based on Chronix"
- Use the name in documentation, tutorials, or articles about Chronix

## Restrictions

You **may not**:

- Use "Chronix" as the primary name of a forked or derived project
- Use the Chronix logo in your own project's branding
- Present your fork or derivative as the official Chronix project
- Use the name or logo in ways that suggest endorsement or affiliation without written permission
- Register "Chronix" or confusingly similar names as trademarks in any jurisdiction

## Naming Derivative Works

If you create a fork or derivative work:

| Acceptable | Not Acceptable |
|------------|----------------|
| "MyProject (fork of Chronix)" | "Chronix Pro" |
| "MyProject, based on Chronix" | "Chronix Enterprise" |
| "A Chronix-compatible tool" | "Open Chronix" |
| "Built on Chronix" | "Chronix 2.0" |

## Logo Usage

The Chronix logo may not be used in derivative projects without explicit written permission. You may use the logo solely to refer to the official Chronix project in documentation or comparative materials.
